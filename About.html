FileManagerLB

FileManagerLB is a file manager focused on bulk file management.
Some features are:
Bulk rename (optional regex usage)
Bulk file marking for future operations
Directory synchronization
Duplicate finder
Snapshots
Standard file operations (move, copy, delete, rename)
